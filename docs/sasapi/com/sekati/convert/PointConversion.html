toPointArray

static public function toPointArray(coords:String):Array

Conversion utility for Polygon.draw(). Accepts condensed string data (usually from XML) & converts to a proper Point array for use with draw.

Parameters

coords(String) - formatting "x,y|x,y|x,y" (e.g. "524,176|599,160|618,252|543,268")

Return

Array - of points 

 Usage:
 	var pointArray:Array = PointConversion.toPointArray( "524,176|599,160|618,252|543,268" );
 	Polygon.draw( polyMc, pointArray, 0xff00ff, 50, 1, 0x00ffff, 100 );
