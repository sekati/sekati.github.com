Class com.sekati.log.OutPanel

  • com.sekati.log.OutPanel

Description

Creates a panel within current swf for live debugging via Out. Can be toggled on/off with hotkey [Shift+Right Arrow] or [Shift + Left Arrow] Can clear panel output with hotkey [Shift + Backspace] or [Shift + Delete] XXX - !!! NOTE: v1.9.0 is under heavy construction - please revert to earlier version in SVN for working version!!! 

 Usage:
 OutPanel.getInstance(640,480);
 -or-
 var out:Out = Out.getInstance();
 out.createPanel(640,480);
