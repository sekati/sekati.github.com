Class com.sekati.net.ExpressInstall

  • com.sekati.net.ExpressInstall

Description

This class invokes the Adobe Flash Player Express Install functionality if an update is required. 

 Flash Usage:
 // first line of code in your fla (before preloader) 
 var expressinstall:ExpressInstall = new ExpressInstall();
 
 swfIN Usage:
 <script type="text/javascript"> 
 	var f = new swfIN("foo.swf", "foo", "640", "480"); 
 	f.useExpressInstall();
 	f.detect(9, "d", "no_flash");
 	f.write();
 </script>
Note: Your Flash movie must be at least 214px by 137px in order to use ExpressInstall.
