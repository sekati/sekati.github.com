Classpublic class JPGEncoder

JPGEncoder provides conversion from Bitmap to valid JPG file format.


Example 
  var jpg : JPGEncoder = new JPGEncoder( 80 );
  var jpgData : ByteArray = jpg.encode( bitmapDataObj );
