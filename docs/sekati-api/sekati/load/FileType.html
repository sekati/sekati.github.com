public static function getLoaderClass(path:String):Class

Detect the appropriate AbstractLoader for the passed in path file type. If no appropriate file type ILoader can be located the default BINARY_LOADER will be used.

Parameters
path:String — of the file to be queued.

Returns
Class — Class reference to the appropriate AbstractLoader sub-class.
