InheritanceSWFLoader AbstractLoader flash.events.EventDispatcher
ImplementsILoader

SWFLoader provides a standard API loader for SWF asset content.


Example 
  var loader : SWFLoader = new SWFLoader( "http://localhost/path/to/movie.swf" );
  loader.addEventListener( Event.INIT, addSWF );
  loader.load( );
  
  private function addSWF(e : Event) : void {
    loader.removeEventListener( Event.INIT, addSWF );
    addChild( loader.content );
  }
