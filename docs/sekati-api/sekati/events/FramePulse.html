Classpublic final class FramePulse
InheritanceFramePulse flash.events.EventDispatcher

FramePulse proxies an ENTER_FRAME event to any non-DisplayObject classes which need to execute code on each frame tick.


Example 
   FramePulse.$.addFrameListener( enterFrame );
   protected function enterFrame( e:Event ):void {
    trace( "pulse" );
   }
